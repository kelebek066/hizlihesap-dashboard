PanelPro — Dashboard şablonu
===========================
Bu küçük proje HTML + CSS + JS ile hazırlanmış profesyonel mor & siyah tema dashboard örneğidir.

Kullanım:
1. Bu klasörü GitHub repoya yükleyin (root'ta index.html bulunsun).
2. GitHub Pages aktif edin (Settings → Pages → main branch → / (root)).
3. Görselleri assets/img içine koyun ve index.html içindeki src yollarını güncelleyin.
